Plywood C++ BASE LIBRARY

Build cross-platform native software in C++

This project is a work in progress and not ready for wide release. Feel free to look around or check back later for updates.

Modules Included

<ply-base.h>OS and Common Primitives
<ply-math.h>2D and 3D Math
<ply-network.h>TCP/IP Networking
<ply-btree.h>B-Trees
<ply-tokenizer.h>Tokenizer
<ply-json.h>JSON
<ply-markdown.h>Markdown
<ply-cpp.h>C++ Parser

Supports all major platforms
